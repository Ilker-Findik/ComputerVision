# CNN Flower

TensorFlow/Keras image classification experiment for the TF Flowers dataset.

## Run

- Execute `cnn.py` from this folder or from the repository root.
- The dataset cache is stored in `Datasets/` at the repository root.
- Training checkpoints are written next to the script and ignored by Git.

## Notes

- The script uses data augmentation and a small CNN baseline.
- Keep large training artifacts out of the repository.
